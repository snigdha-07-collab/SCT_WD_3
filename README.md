# 🎮 Tic-Tac-Toe Web Application

## 📌 Overview

This project is a responsive Tic-Tac-Toe Web Application built using HTML, CSS, and JavaScript. It allows users to play against another player or an easy AI. The application features a modern interface, live score tracking, winner detection, and responsive design.

## ✨ Features

- 🎮 Player vs Player Mode
- 🤖 Player vs Computer Mode
- 🏆 Live Scoreboard
- 🔄 Restart Game
- 🗑️ Reset Scores
- ✅ Winner Detection
- 🤝 Draw Detection
- 🎨 Winning Cell Highlight
- 📱 Responsive Design
- ⚡ Fast and Interactive UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or run it using Live Server in VS Code.
4. Enjoy playing!
